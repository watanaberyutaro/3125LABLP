# 3125 LAB - 公式ランディングページ

> 痒いところを、"仕組み"で掻く研究所。

## 🧪 プロジェクト概要

3125 LABの公式LPです。企業が抱えるニッチな業務課題をAIとローコードで実験的に解決する開発スタジオのブランドサイトです。

### ✨ 特徴

- **研究所風デザイン**: ラボをテーマにしたモダンでプロフェッショナルなデザイン
- **ドット調イラスト**: 手書き風のSVGイラストを随所に配置
- **スムーズアニメーション**: Framer Motionによる洗練されたアニメーション
- **レスポンシブ対応**: モバイル・タブレット・デスクトップ完全対応
- **SEO最適化**: Meta タグ、OGP設定済み

## 📧 お問い合わせフォーム設定

### メール送信先

お問い合わせフォームから送信されたメールは、環境変数で設定したメールアドレスに届きます。

### SMTP設定（本番環境）

メール送信機能を有効にするには、環境変数の設定が必要です。

1. `.env.local` ファイルを作成（`.env.local.example` を参考に）

```bash
cp .env.local.example .env.local
```

2. 環境変数を設定

```env
# SMTPサーバー設定
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password

# 送信元アドレス
SMTP_FROM=your-email@example.com

# お問い合わせ受信先アドレス
CONTACT_EMAIL=your-contact@example.com
```

### Gmail使用時の設定手順

1. Googleアカウントで「アプリパスワード」を生成
   - https://myaccount.google.com/apppasswords
2. 生成したパスワードを `SMTP_PASS` に設定
3. `SMTP_USER` にGmailアドレスを設定

### 開発環境での動作

開発環境では環境変数が未設定でもエラーは発生しませんが、実際のメール送信は行われません。本番環境でのみ環境変数を設定してください。
